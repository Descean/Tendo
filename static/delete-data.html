Suppression des Données Utilisateur

Dernière mise à jour : 23 mars 2026

Conformément à la loi béninoise (loi n° 2017-20 portant code du numérique) et au RGPD, vous avez le droit de demander la suppression complète de vos données personnelles à tout moment.

1. Données concernées

La suppression de votre compte entraîne la suppression définitive de :

  • Votre profil — numéro de téléphone, nom, entreprise
  • Vos préférences — secteurs, régions, sources de veille
  • Votre historique — messages échangés avec Tendo
  • Vos notifications — historique des alertes reçues
  • Votre abonnement — données de souscription (les transactions FedaPay restent conservées chez FedaPay pour des raisons légales)

2. Comment demander la suppression

Methode 1 : Via WhatsApp (le plus simple)

  1. Ouvrez la conversation WhatsApp avec Tendo
  2. Envoyez le message "Supprimer mon compte"
  3. Confirmez votre demande quand Tendo vous le demande
  4. Vos données seront supprimées sous 48 heures

Methode 2 : Par email

  1. Envoyez un email à privacy@shiftup.bj
  2. Indiquez votre numéro de téléphone WhatsApp dans l'email
  3. Précisez "Demande de suppression de données" en objet
  4. Nous traiterons votre demande sous 72 heures maximum

3. Délais de traitement

  • Via WhatsApp : suppression sous 48 heures
  • Via email : suppression sous 72 heures
  • Vous recevrez une confirmation par WhatsApp ou email une fois la suppression effectuée
Attention : La suppression de vos données est irréversible. Si vous avez un abonnement actif, celui-ci sera annulé sans remboursement pour la période en cours. Vous pourrez créer un nouveau compte ultérieurement si vous le souhaitez.

4. Données conservées après suppression

Certaines données peuvent être conservées après la suppression de votre compte :

  • Transactions financières — conservées 5 ans conformément aux obligations comptables
  • Logs techniques anonymisés — conservés 3 mois pour la sécurité du service

Ces données résiduelles sont anonymisées et ne permettent pas de vous identifier.

5. Contact

Pour toute question relative à la suppression de vos données :

  • WhatsApp : Envoyez "Support" au numéro Tendo
  • Adresse : SHIFT UP — Cotonou, Bénin
